Powerful Features

Everything you need for modern financial services through WhatsApp

Savings Management

Check balances, make deposits, withdrawals, and transfers directly through WhatsApp messages.

Bitcoin Integration

Real-time Bitcoin prices, portfolio tracking, and cryptocurrency management.

Advanced Chama Management

Create chama groups, contribute to shares, track member contributions, and view detailed shares balance across all your chama investments.

Bank-Grade Security

End-to-end encryption, rate limiting, and comprehensive security audits.

Voice Messages

Send voice commands and receive audio responses for hands-free operation.

High Performance

Built with Rust for maximum speed, reliability, and memory efficiency.

Bot Commands

Simple, natural language commands that work seamlessly

General Commands

help Show available commands
balance Check savings and BTC balance

Financial Operations

deposit 100 USD Make a deposit
withdraw 50 KES Make a withdrawal
transfer 25 USD +254712345678 Transfer to another user

Bitcoin Services

btc Get current Bitcoin price
bitcoin price Alternative BTC price command

Chama Management

chama View your chama groups and members
create chama My Group Create a new chama group
contribute chama CH123 50 USD Contribute to chama shares
shares balance View your chama shares balance
shares balance CH123 View shares for specific chama

Personal Savings

savings View detailed savings information

Example Usage

Send Message 
curl -X POST http://localhost:8080/send \
  -H "Content-Type: application/json" \
  -d '{
    "to": "+254712345678",
    "message": "Hello from BitSacco Bot!"
  }'
Health Check 
curl http://localhost:8080/health

Quick Deployment

Get up and running in minutes with our deployment options

Docker

Containerized deployment with Docker

docker run -d \
  --name bitsacco-bot \
  -p 8080:8080 \
  --env-file .env \
  bitsacco-whatsapp-bot:latest

Railway

One-click deployment to Railway

npm install -g @railway/cli
railway login
railway deploy

Manual

Build and run from source

cargo build --release
./target/release/bitsacco-whatsapp-bot
